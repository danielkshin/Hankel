Pitches

Changing pitches

Use the menu below the planner to change the pitch type. Click on the desired pitch to change the pitch in the planner.

Players

Adding players

Use the menu on the left of the planner to add players. Click on the desired player then click on the desired location in the planner.

Equipment

Adding equipment

Use the green menu on the right of the planner to add equipment. Click on the desired equipment then click on the desired location in the planner.

Markup

Use the black menu on the right of the planner to add markup.

Color

Click the square then adjust the color using the marker, the eyedrop tool, or custom RGB values.

Draw

Click the icon then draw by clicking and dragging in the planner.

Text

Click the icon then click on the desired location in the planner. Type the desired text into the following prompt and click OK to add the text.

Line / Dashed Line

Click the or icon then click and drag in the planner to add a line or dashed line from the initial clicking point to the point of release.

Line / Dashed Arrow

Click the or icon then click and drag in the planner to add an arrow or dashed arrow from the initial clicking point to the point of release.

Line / Dashed Curved Arrow

Click the or icon then click and drag in the planner to add a curved arrow or dashed curved arrow from the initial clicking point to the point of release. The head of the curved arrow will face the point of release.

The curved arrow will automatically change between an inner curve and an outer curve. To change the curve manually, hold the SHIFT key down while dragging the mouse.

Tools

Use the black menu on the right of the planner to use tools.

Move

Click the icon then click and drag the desired object (text, players, equipment) to move it.

Delete

Click the icon then click the desired object to remove it from the planner. To delete multiple objects at once, click and drag the mouse over desired objects.

Save as Image

Click the icon then name the image and click save or OK to save the plan as an image.

Save as JSON (experimental)

Click the icon then name the JSON and click save or OK to save the plan as a JSON.

Load JSON (experimental)

Click the icon then select the JSON file and click Open to load the planner from the JSON file.

Notes

Use the text area below the planner to add notes to the plan. Click the text box then type notes. Use the check box below the text box to select whether to include the notes when saving the plan as an image.
